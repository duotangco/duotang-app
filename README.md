# Duotang

**Tab manager and workspace extension for Chrome.** Free, local-first, no account required.

[Install from the Chrome Web Store](https://chromewebstore.google.com/detail/duotang/apglpidnigbjhacfhnfklknpadjfodaj) · [duotang.co](https://duotang.co) · [Privacy Policy](https://duotang.co/privacy)

---

## What it does

Duotang organizes your browser tabs into named, color-coded workspaces. Switching between workspaces does not close or reload your tabs — the Gmail draft you were writing, the Figma file you had open, the AI conversation mid-session — all survive a workspace switch untouched.

Most tab managers that call themselves "workspace tools" save and restore URLs. Duotang keeps the tabs themselves alive.

**Core features:**

- **Workspaces** — named, color-coded tab containers. Switch instantly; nothing reloads.
- **Find & Switch** — search across all your tabs, workspaces, history, downloads, and clippings from any page (Ctrl+Shift+Space).
- **Peek** — open any link in an overlay alongside your current page. Alt+click any link, or right-click for the Peek pill.
- **Highlights** — select text on any page. Duotang recognizes currencies, units, dates, addresses, colors, and reference codes and surfaces actions inline — no new tab needed.
- **Workbench** — a per-workspace side panel with notes (Pad), an embedded map (Maps), and workspace-scoped file access (Files).
- **Search+** — workspace context overlaid on Google, Bing, DuckDuckGo, and Brave results. See which workspace you visited a result from, with last-visit timestamp.
- **One Window mode** — single-window workspace switching. Inactive workspace tabs stay alive in a hidden background window; switching is instant.
- **Multi-window mode** — each open workspace lives in its own Chrome window. Multiple workspaces alive simultaneously.
- **Minimize tabs** — tuck a tab away with scroll position captured; restore it later from the minimize bar.
- **Workspace-scoped restore** — Ctrl+Shift+Z restores the most recently closed tab in the current workspace, not the globally last-closed tab.

## Privacy

- All data stored locally in `chrome.storage.local`. Nothing leaves your device for storage purposes.
- No telemetry, no analytics, no tracking pixels, no third-party scripts.
- No account required. No user IDs or device fingerprints of any kind.
- Crash reports are opt-in and fully anonymous.
- Network access is limited to user-initiated fetches: currency rates, Wikipedia summaries, map tiles, and scripture lookups via public APIs. Every network-using feature is individually toggleable. A Paranoid mode blocks all outbound calls entirely.

Full privacy policy: [duotang.co/privacy](https://duotang.co/privacy)

## Compatibility

Chrome 121+, Edge, Brave, Vivaldi, Opera, and other Chromium-based browsers. Chrome only — no Firefox support.
