# Contributing

Duotang is not open source — the source code is not available for contribution. Pull requests cannot be accepted.

**Bug reports and feature requests are very welcome** via [GitHub Issues](https://github.com/duotangco/duotang-app/issues). Please use the issue templates so reports arrive with the information needed to reproduce and investigate.

For security vulnerabilities, see [SECURITY.md](SECURITY.md) — please do not file those publicly.

For anything else: human@duotang.co.
